Portfolio

BLUR App
  • Git Repository: https://github.com/ICiovica/Blur
    • The app is designed to solve the perennial problem of parents who want to protect their children by blurring their faces in photos that are shared on social media. For simplicity, the app will automatically detect the faces in the imported photo, with the possibility for the user to choose which face will be blurred. Also, the app will allow the user to set the blur intensity, to share the photo directly from the app, or to save it in the Photo Library.
    • Vision framework to detect all the faces in a photo.
    • CoreImage framework to implement dedicated filters for blurring.
    • CoreML framework to implement a trained model which detects if there is a child in the image coloring its border red, or an adult coloring its border with green.
    • UIKit framework to add Timer, UITapGestureRecognizer, UIImagePickerController, UIGraphicsImageRenderer, #selectors, etc.
    • CLTypingLabel from cocoapods.org, which is a lightweight UILabel subclass with character by character typewriter like animation in Swift.
